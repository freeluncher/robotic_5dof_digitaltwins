# apps/web

Frontend React + Vite untuk digital twin robotic arm 5 DOF.

## Fokus

- UI operasional dan visualisasi 3D.
- State management dengan Zustand.
- Unit dan component test dengan Vitest.

## Testing

- `npm test` menjalankan suite Vitest.
- `npm run test:watch` menjalankan mode watch untuk TDD.

## Catatan

Gunakan folder ini untuk semua test frontend yang memverifikasi perilaku UI, state, dan interaksi real-time sebelum implementasi diperluas.
